おうち家計簿

収入・支出・カテゴリ・カレンダー・グラフ・貯金額を管理できる、 シンプルなWindows向け家計簿アプリです。

Windows版をダウンロード

主な機能

  • 収入・支出の登録
  • カテゴリ別管理
  • 親カテゴリ・子カテゴリの設定
  • カレンダーでの日別確認
  • 検索と期間指定
  • カテゴリ別グラフ
  • 月別収支グラフ
  • 貯金額推移の確認
  • 家計簿CSV保存
  • PayPay CSV取込
  • バックアップ作成・復元
  • 全データ初期化

使い方

  1. ZIPファイルをダウンロードします。
  2. ZIPファイルを展開します。
  3. フォルダ内の kakeibo.exe をダブルクリックして起動します。

kakeibo.exe だけを取り出さず、 data フォルダや .dll ファイルも同じフォルダに置いた状態で起動してください。

データの保存について

入力した家計簿データは、お使いのPC内に保存されます。 開発者へ家計簿データが送信されることはありません。

データ消失に備えて、設定画面から定期的に 「CSV保存」または「バックアップ作成」を行ってください。

CSVについて

CSVはUTF-8形式で保存されます。 Excelで文字化けする場合は、「データ」→「テキストまたはCSVから」読み込んでください。

注意事項

このアプリは家計管理を補助するためのツールです。 金融・税務・投資などの専門的な助言を行うものではありません。

利用者の環境や操作状況によって、データが消失する可能性があります。 重要なデータは、必ず利用者自身でバックアップを作成してください。

対応環境

Windows 10 / Windows 11 を想定しています。
